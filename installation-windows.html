Windows Installation
Some notes on installation for Windows, using Windows 10 x64 and Microsoft Visual Studio 2015.

1. Downloads and installation applications

  • CMake (tested with 3.10.0)
  • Git for Windows (tested with 2.15.1)
  • QT5 (tested with 5.9.3)
  • Python (tested with 2.7.13)
  • Boost (tested with 1.65.1)
Install applcations with default settings, unpack the boost in C:\ and create the environment variables: BOOST_ROOT with value C:/boost_1_65_1,
Qt5_DIR with value C:/Qt/5.9.3/msvc2015_64/lib/cmake/Qt5.

2. Installing VTK

Open Command Prompt by pressing Windows+X to open the Power Users menu, and then click "Command Prompt (Admin).".

Run the commands for getting ParaView and VTK sources on C:\paraview,C:\paraview\VTK:
cd ../..
"C:\Program Files\Git\mingw64\bin\curl.exe" -o "paraview.tar.gz" "https://www.paraview.org/paraview-downloads/download.php?submit=Download&version=v5.4&type=source&os=all&downloadFile=ParaView-v5.4.1.tar.gz"
"C:\Program Files\Git\usr\bin\gzip.exe" -d paraview.tar.gz
"C:\Program Files\Git\usr\bin\tar.exe" xvf paraview.tar
move c:\ParaView-v5.4.1 paraview
del paraview.tar

Run the commands for opening CMake with VTK:
cd paraview/VTK
mkdir build
cd build
cmake-gui ../

Configure VTK (select generator "Visual Studio 14 2015 Win64" for x64-bit application):
 · CMAKE_CONFIGURATION_TYPES=Release
Generate and open project.
Make sure that INSTALL is the selected target. Build it and close CMake.
Change the environment variable PATH to include the path: C:\Program Files\VTK\bin

3. Download TTK

Run the commands for getting TTK sources on C:\ttk:
cd ../../..
git clone https://github.com/topology-tool-kit/ttk.git

4. Patching ParaView

Run the commands for patching ParaView on x86 installation
cd c:/ttk/paraview/patch
patch-paraview-msvc-x86.cmd c:/paraview
Run the commands for patching ParaView on Windows
patch-paraview-msvc.cmd c:/paraview
Run the commands for adding to the ParaView source tree the complete set of TTK features
patch-paraview-5.4.1.sh c:/paraview
cd ../../..

5. Installing ParaView

Run the commands for opening CMake with ParaView:
cd paraview
mkdir build
cd build
cmake-gui ../

Configure ParaView:
 · CMAKE_CONFIGURATION_TYPES=Release
 · PARAVIEW_ENABLE_PYTHON=ON
 · PARAVIEW_INSTALL_DEVELOPMENT_FILES=ON
Generate and open project.
Make sure that INSTALL is the selected target. Build it and close CMake.
Create the environment variable PARAVIEW_BIN with value C:\Program Files\ParaView\bin

6. Installing TTK

Run the commands for opening CMake with TTK:
cd ../../ttk
mkdir build
cd build
cmake-gui ../

Configure TTK:
 · CMAKE_CONFIGURATION_TYPES=Release
 · TTK_ENABLE_OPENMP=OFF
Generate and open project.
Make sure that INSTALL is the selected target. Build it, close CMake and Command Prompt.
